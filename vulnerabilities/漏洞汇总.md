# 漏洞 PoC 演示汇总

本目录包含跨语言（Rust -> C）供应链漏洞的概念验证（PoC）演示。

## 可用 PoC 列表

### 1. CVE-2024-28757 (libexpat XML 实体膨胀)
- **位置**: `CVE-2024-28757_expat/`
- **描述**: XML 实体膨胀攻击（Billion Laughs），通过 C 组件间接调用触发。
- **运行**: `./CVE-2024-28757_expat/run_poc.sh`
- **说明文档**: 请参阅该目录下的中文说明（如有）。

### 2. CVE-2023-50472 / CVE-2023-50471 (cJSON 模拟漏洞)
- **位置**: `CVE-2023-50472_cjson/`
- **描述**: JSON 压缩（Minify）过程中的堆缓冲区越界读取/溢出，由未闭合的注释引起。
- **运行**: `./CVE-2023-50472_cjson/run_poc.sh`
### 3. CVE-2024-28757 (Chain Version)
- **位置**: `CVE-2024-28757_Chain/`
- **描述**: CVE-2024-28757 的多跳（Chain）变体，演示更复杂的调用链。
- **运行**: `./CVE-2024-28757_Chain/run_poc.sh`

### 4. Indirect Libxml2
- **位置**: `Indirect_Libxml2/`
- **描述**: 演示通过 C 组件间接调用 libxml2 的场景。
- **运行**: `./Indirect_Libxml2/run_poc.sh`
